# **Scripts**

### Running the backend-code

---

**clone the repo**

```bash
git clone https://github.com/Sam44323/zap-backend.git
```

---

**Install the dependencies**

```bash
yarn
```

---

**populate the env's**

```bash
cp .example.env .env

```

---

**Running the dev-server**

```bash
yarn start
```

**Running the tests**

```bash
yarn test
```
